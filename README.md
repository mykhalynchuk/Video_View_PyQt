PyQt6 Video Stream Application

Опис

Це проста десктопна програма на Python, яка використовує PyQt6 та OpenCV для відображення відео з камери в реальному часі.

Користувач може:

• запускати відеопотік

• зупиняти відео

• перемикатися між доступними камерами


Вимоги

Перед запуском переконайтеся, що у вас встановлено:

• Python 3.8+

• pip (менеджер пакетів Python)


Встановлення залежностей

Виконайте в терміналі:

pip install opencv-python PyQt6


Запуск програми

Збережіть код у файл, наприклад: main.py

Запустіть програму: python main.py


Як користуватися

• Натисніть Start щоб почати відео

• Натисніть Stop щоб зупинити

• Оберіть камеру зі списку для перемикання


Як це працює

• Використовується QThread для роботи з відео в окремому потоці

• OpenCV отримує кадри з камери

• PyQt6 відображає кадри у QLabel

• Сигнали (pyqtSignal) передають зображення між потоками


Примітки

• Індекси камер (0, 1, ...) можуть відрізнятися залежно від вашого пристрою

• Якщо камера не відкривається — перевірте доступ до неї

• Не запускайте кілька потоків одночасно
